## ✨ 项目特色
- 🎯 **竞赛导向**：专为ACM-ICPC/CCPC/NOI/NOIP等算法竞赛设计
- 📖 **全面覆盖**：从基础数据结构到高级算法，一应俱全
- ⚡ **高性能实现**：优化的C++17代码，追求极致效率
- 🧩 **模块化设计**：每个算法独立成文件，即插即用
- 📝 **详细注释**：中英文注释，思路清晰，易于理解

## 🛠️ 快速开始
### 环境要求
- C++17 或更高版本
- GCC 9.0+ / Clang 10.0+ / MSVC 2019+
### 编译运行
```bash
# 克隆仓库
git clone https://github.com/yourusername/SUSTech-OI_wiki-DSAA.git
cd SUSTech-OI_wiki-DSAA
# 编译单个文件
g++ -std=c++17 -O2 -Wall data_structures/basic/stack.cpp -o stack
# 运行测试
./stack
```
### 使用模板
```cpp
#include "templates/fast_io.cpp"
#include "data_structures/basic/stack.cpp"
using namespace std;
int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    // 你的代码
    return 0;
}
```
## 🎯 核心内容
### 数据结构
- **基础结构**：栈、队列、链表、哈希表
- **树结构**：线段树、树状数组、字典树、平衡树
- **图结构**：邻接表、邻接矩阵、并查集
- **高级结构**：可持久化数据结构、珂朵莉树
### 算法
- **排序**：快速排序、归并排序、基数排序
- **搜索**：二分搜索、三分搜索、深度优先搜索、广度优先搜索
- **图算法**：最短路、最小生成树、网络流、二分图匹配
- **动态规划**：线性DP、区间DP、树形DP、数位DP
- **数学算法**：数论、组合数学、矩阵快速幂、FFT
